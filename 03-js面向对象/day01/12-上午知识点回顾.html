1.JS的组成: DOM: 文档对象模型 BOM: 浏览器对象模型 ECMAScript: 规定了JS的核心语法 2.基本数据类型和复杂数据类型 基本数据类型: string number boolean undefined null 复杂数据类型: Object Date Math Array Function String Number Boolean 如何判断一个数据的类型: typeof 语法: typeof 数据(变量) 返回值类型: 字符串 注意点: 如果是复杂数据类型使用typeof是object,但是函数除外,函数是function 如何判断对象是否是一个数组: Array.isArray(对象) 3.等于和全等于 = 赋值: 把右边存储的信息复制一份给左边 == 等于: 判断右边数据是否与左边的数据相等 如果值类型和值类型进行判断,判断的就是具体值 如果是引用类型和引用类型进行判断,判断的是指向具体数据的地址 如果值类型和引用类型进行判断 把引用类型进行隐式转换,取出具体的值,再与值类型进行判断 === 全等于: 除了判断判断右边数据是否与左边的数据相等之外还会去判断类型 4.关系运算符: > < >= <= 5.逻辑运算符 逻辑非: 取反 语法: !表达式 结果: 是布尔类型 逻辑或 语法: 表达式1 || 表达式2 结果: 不是布尔类型 如果表达式1为真,返回表达式1 如果表达式1为假,返回表达式2 逻辑与 语法: 表达式1 && 表达式2 结果: 也不是布尔类型 如果表达式1为真,返回表达式2 如果表达式1为假,返回表达式1 6.值类型和引用类型 值类型存储的是具体的数据 引用类型存储的是指向具体数据的内存地址 7.值类型和引用类型赋值 赋值(=) : 把右边存储的信息复制一份给左边 值类型赋值: 把右边存储的信息(具体的数据)复制一份给左边 特点: 值类型赋值后,互不影响 引用类型赋值: 把右边存储的信息(指向具体数据的内存地址)复制一份给左边 特点: 引用类型赋值后,会共享同一份数据,修改其中一个对象的属性,会影响另外一个对象 8.值类型和引用类型在函数中的使用 实参: 实际的参数,函数调用时传入的参数 形参: 占位用的,函数调用前没有值 函数调用: 默认情况下会把实参的值赋值给形参 值类型作为函数的参数(值传递): 实参和形参是独立的,互不影响 引用类型作为函数的参数(地址传递): 实参和形参共享同一份数据,互相有影响 9.对象的动态特性 一个对象在创建出来之后,可以执行的操作: 添加 | 删除 | 修改 语法: 1.点语法 2.[]语法: []中必须是字符串 注意点: 1.如果给一个对象的属性赋值,如果这个对象有这个属性就是修改,如果没有这个属性就是添加 2.使用点语法很方便,但是有时候使用点语法无法来获取对象的属性(属性有空格 | 变量的值是字符串只能使用[]语法) 10.in关键字 1.for..in遍历对象(key-value键值对)中的属性 2.判断对象是否有指定的属性 语法: 属性 in 对象 注意点: 属性必须是字符串 返回值: 布尔类型 注意点: 1.操作数据的时候不是操作数据的元素,而是操作数据的下标 2.判断数组中是否有某个元素? 1.遍历整个数组,获取每个元素判断对应的值 2.indexOf -> 元素在数组中的下标,如果没有 -> -1 11.delete 1.删除对象中的属性 2.删除未使用var声明的变量 注意点: 1.这个结果是有返回值,类型为布尔类型,true代表删除成功,false代表删除失败 2.不能删除使用var声明的变量 3.但是可以删除直接定义在window上的属性
