函数的隐藏参数 arguments: 函数调用的时候,会将实参保存到arguments中,arguments不是一个数组,是一个类似于数组的对象(Arguments) 1.实参的个数 > 形参的个数,多余的实参可以通过arguments获取 2.实参的个数 < 形参的个数,会依次给形参赋值,多余的形参的值为undefined length: 获取实参的个数: arguments.length 获取形参的个数: 函数名.length this: 函数中指向的对象 1.函数作为对象的方法调用: this -> 这个对象 2.函数作为普通函数调用: this -> window 3.函数作为构造函数调用: this -> 内部创建的对象 4.(call | apply): this -> 第一参数
